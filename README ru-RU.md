# Vardone
## Описание
Vardone - это приложение для мгновенного обмена сообщениями между пользователями.

В приложении реализованы системы:
- акаунтов
- серверов
- друзей

## Использованные технологии
В разработке Vardone были использованы технологии:
- ASP.NET Web API
- Rest API
- WPF
- EntityFramework
